# Cursed Racing

## Build

### Release
```
make all
```

### Debug
```
DEBUG=1 make all
```

## CLI Args
```
--help        Print help message
--rand=<int>  Initialize random numbers generator (essentially your track number)
```

## Gameplay

Use cursor keys to set the direction.

Use "-" and "+" (or "=") to increase or decrease speed.

Press "Enter" to make a turn.

Press "q" to exit.
